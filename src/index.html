How to get a free API key

  3. Paste it above

Connect Google Photos

Import all geotagged photos from your Google Photos library using a Takeout export. Only coordinates and tiny thumbnails are stored — no full photos.

How to export your Google Photos (~2 min)
  1. Go to takeout.google.com
  2. Click Deselect all, then check only Google Photos
  3. Click Next step → choose .zip format → Create export
  4. Once ready, download and unzip the archive
  5. Click the button below and select the Google Photos folder from the unzipped archive
or pick specific photos

  3. Go to APIs & Services → Library, search for Photos Picker API and enable it
  4. Go to APIs & Services → Credentials
